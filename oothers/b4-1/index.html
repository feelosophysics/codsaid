About

기본 동작 원리를 직접 구현합니다

이 포트폴리오는 외부 프레임워크 없이 브라우저가 기본으로 제공하는 기능만 사용합니다. 버튼을 누르면 이벤트가 발생하고, JavaScript는 상태를 바꾸며, 바뀐 상태에 맞게 DOM을 다시 업데이트합니다.

이런 흐름은 React, Vue 같은 프레임워크를 배울 때도 그대로 이어집니다. 그래서 이 프로젝트는 보기 좋은 페이지를 만드는 것뿐 아니라, 웹 앱이 어떤 순서로 생각하고 움직이는지 익히는 연습입니다.
